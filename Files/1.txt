Uma classe é um modelo que define a forma de um objeto.Ela especifica tanto os dados quanto o código que operará sobre elas.Java usa uma especificação de classe para construir objetos.

Os objetos são instâncias de uma classe.

Logo,uma classe é um conjunto de planos que especifica como construir um objeto.É importante deixar uma coisa bem clara,uma classe é uma abstração lógica.Só quando um objeto dessa classe é criado é que existe uma representação física dela na memória.

Outro ponto:lembre-se de que os métodos e variáveis (atributos) que compõem uma classe são chamados membros da classe.Os membros da classe também são chamados de variáveis de instância.

- Forma geral de uma classe -

Uma classe é criada com o uso da palavra chave class.

Exemplo:  
           class nome_da_classe {
           // declara atributo
              tipo var1;
              tipo var2;
           // declara métodos
              tipo método(parâmetros){
              // corpo do método
              }
            }
