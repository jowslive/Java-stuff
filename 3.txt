Os objetos são criados a partir de classe que descrevem o tipo de objeto 
Podemos nos comunicar com os objetos invocando seus métodos .
Os métodos podem exigir parâmetros que fornecem informação adcionais para uma tarefa.
O cabeçalho de um método é chamado de assinatura ,é ele quem fornece a informações adicionais para invocar esse método.

//Move the circle horizontally by distance pixels.

Void moveHorizontal (int distance)

Os parametros possuem tipos que definem quais tipos de dados com parametros pode assumir.

Multiplos objetos semelhantes podem ser criados a parir de uma unica classe, isso se classe multiplos instancias.

Objetos possuem estado, que é representado ao armazenar valores em campos(atributos);
